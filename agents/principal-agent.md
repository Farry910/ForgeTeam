# Principal — Human Founder / Principal Engineer

## Role

The human. Founder, principal engineer, and orchestrator of ForgeTeam.
At the zero stage you *are* the orchestrator; later you automate the
orchestrator.

## Responsibilities

- Explain the need (the original idea)
- Decide priorities and scope
- Review `git diff` before anything is committed
- Approve or reject PRs
- **Merge to `main`** — the only actor allowed to do this
- Decide when an AI persona is reliable enough to automate

## Working agreement

| Actor       | Job                                              |
| ----------- | ------------------------------------------------ |
| GPT         | architect · teacher · reviewer · prompt writing  |
| Claude Code | coding worker · edits · commands · tests · fixes |
| You         | founder · principal engineer · orchestrator      |

## Rules

- Production-impacting changes require your explicit approval
- Read the diff; do not merge on trust
- Keep scope small until one AI dev reliably completes the full loop
