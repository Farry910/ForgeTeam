# ForgeTeam — Hybrid Workspace MVP

## Goal

Build a local-first hybrid software workspace where AI developers behave like
human teammates. ForgeTeam is the workflow shell that makes one believable AI
developer move a task from idea → code → test → PR like a real engineer.

## MVP Scope

The first version supports:
- task creation
- AI developer profiles
- status tracking
- work logs
- GitHub PR links
- human approval

## Non-goals

The first version will **not** include:
- Slack integration
- Jira integration
- autonomous production deploy
- paid cloud infrastructure
- real multi-agent autonomy

## Success metric for v0.1

A stranger can look at this repo and believe:

> "This is the beginning of an AI software team workspace."

Build the workflow shell first. Build it so it can later become real.
